\# Privacy Policy / Политика конфиденциальности



This bot does not store any personal user data except what is necessary for its operation (e.g., user IDs for server commands).

No data is sold or shared with third parties.



Этот бот не хранит личные данные пользователей, кроме необходимых для работы (например, ID пользователей для команд на сервере).

Данные не продаются и не передаются третьим лицам.
