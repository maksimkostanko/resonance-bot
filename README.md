\#Resonance Music Bot



\*\*English:\*\*

Resonance is a Discord music bot designed to play music, manage queues, and provide an interactive music experience for servers.

It uses Discord API and supports role management for smooth operation.



\*\*Русский:\*\*

Resonance — музыкальный бот для Discord, который воспроизводит музыку, управляет очередью треков и предоставляет интерактивный опыт для серверов.

Бот использует API Discord и поддерживает управление ролями для корректной работы.
